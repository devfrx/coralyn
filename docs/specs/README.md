# Specs di progettazione

Questa cartella contiene gli **spec di design** dei moduli, prodotti dal processo di
brainstorming prima dell'implementazione.

Convenzione di naming: `YYYY-MM-DD-<topic>-design.md`.

Flusso per ogni modulo: **brainstorming → spec (qui) → piano di implementazione →
implementazione**. Le decisioni architetturali emerse durante uno spec vengono
promosse ad [ADR](../architecture/decisions/).
