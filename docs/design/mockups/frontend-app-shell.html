Ombrellone 8

Settore Centro · Fila 2 · Normale

Rifiniture di produzione: token CSS (design-system.md), Inter + tabular-nums su date/prezzi/etichette; etichette con ink scuro per-stato (contrasto WCAG AA, non testo bianco); ogni cella è un <button> con focus-visible a doppio anello (vedi il 14) e aria-label testuale (vedi l'8) — lo stato non dipende dal solo colore; selezione (anello teal + alone) distinta dal focus; hover sobrio; reveal scaglionato per fila (con prefers-reduced-motion). Ancoraggio: etichette fisiche reali (buco al 19, "20bis", ADR-0016); Tipologia via marcatore a icona modulare; settore Speciali per le palme; cella slot-aware (l'8: mattina prenotata / pomeriggio libero, ADR-0013); drawer con stato pagamento (ADR-0011) e le 3 azioni dei flussi (flows §2). Aggiornare quando cambia la direzione UI (ADR-0009).
